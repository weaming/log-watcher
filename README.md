# log-watcher

`tail -f` your logs. Only for systems support inotify.

## Install

```
git clone https://github.com/weaming/log-watcher
cd log-watcher
make install
```

or [cross-complilation](https://crystal-lang.org/reference/syntax_and_semantics/cross-compilation.html)

## Usage

```
log-watcher [<port>]
log-watcher --help
```
